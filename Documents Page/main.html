IO

To import data from a csv, use the following:

A = importdata(filename) Col1Header = A.colheaders{1, 1} Col1Data = A.data(:, 1)

Title

To add a title, use the following:

title("Task 5C: Phasor");

Legend

To make a legend, use the following:

legend({"V1","V2","V3","I1","I2","I3"});

Color

To change the color, use the following:

ca(1).Color="r";

Possible colors are:

  • 'r' - Red
  • 'g' - Green
  • 'c' - Cyan
  • 'm' - Magenta
  • 'y' - Yellow
  • 'k' - Black
  • 'w' - White
  • 'none' - N/A

Phasor

To make a phasor diagram, use the following:

phsr1 = [4.008 59.935; 7.086 27.616; 7.75 14.37]; phsr2 = [5.99 -30.07; 3.716 -62.32; 1.946 -75.92]; figure(1) compass(phsr1(:,1).*cosd(phsr1(:,2)), phsr1(:,1).*sind(phsr1(:,2))) figure(2) compass(phsr2(:,1).*cosd(phsr2(:,2)), phsr2(:,1).*sind(phsr2(:,2)))
